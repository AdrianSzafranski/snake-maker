Rules

  1. The user controls the snake.

    Unless the snake is too fast, in which case the snake controls the user's

  2. The snake moves automatically, the user can change its direction of movement.

    "It's not what happens to us, but how we react to it that matters for our destiny" ~Epictetus

  3. The snake dies when it hits an obstacle, itself or an opponent.

    If the user is a minor, the snake just falls asleep

  4. Food changes the snake's properties and the game score.

    A snake eats to grow, but never to get fat. It's fun to be a snake

  5. The way to win depends on the game mode.

    I have run out of ideas for comments

Standard food:

Regular food: +1 length; +1 speed

Speed food: +2 length

Speed food: +2 speed

Special food:

Fortune food: +5 score

Curse food: -5 score

Unknown food: from -10 to 10 score

Game controls

Snake control

Use the arrow keys on your keyboard.

Pause

Use the 'p' key on your keyboard.

Restart

Use the 'r' key on your keyboard.
